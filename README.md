# Tic-Tac-Toe Game

A simple Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**.

## Features

* Two-player gameplay (X and O)
* Winner detection
* Draw detection
* Restart/New Game option
* Clean and responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Open the game in your browser.
2. Player X starts first.
3. Players take turns clicking on empty cells.
4. The first player to get three marks in a row, column, or diagonal wins.
5. If all cells are filled and no player wins, the game ends in a draw.
